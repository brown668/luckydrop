# LUCKY DROP — 游戏设计文档

**版本：** 3.0  
**日期：** 2026-04-12  
**平台：** 移动端竖屏网页游戏（390px 基准）  
**融合公式：** Plinko 物理弹球 × 幸运宝箱盲盒

---

## 一、产品定位

一句话定义：竖屏 Plinko，球从穹顶落入 12 行钉子板，落入底部 15 个倍率槽结算，中间两个幸运宝箱随机爆出 1-100 倍，三种难度满足不同风险偏好，目标 RTP 97.5%。

## 三、钉子板几何参数

| 参数 | 值 |
|-----|----|
| 宽度 W | 390px |
| 行数 ROWS | 12 |
| 槽数 SLOTS | 15 |
| 钉子间距 d | 26px（W/SLOTS） |
| 行高 rowH | 26px |
| 钉子半径 pegR | 5px |
| 顶部留白 padT | 54px（含穹顶） |
| 钉子排列 | 行 r 有 (r+3) 颗钉子，居中对齐 |
| 壁角 slope | d/(2×rowH) = 0.5 |
| 穹顶半径 domeR | 46px（= 7d/4） |

球路从穹顶中心落下，经 12 行钉子扩散，落入 15 槽之一，概率分布近似二项 B(14, 0.5)，中心槽概率约 20.9%，边缘槽约 0.006%。

## 五、幸运宝箱（Mystery Chest）

- 位置：槽位 2 和槽位 12（对称）
- 外观：金色 SVG 宝箱图标，持续金光脉冲动画
- 机制：球落入宝箱槽时，随机生成 **1-100 倍**（整数）
- 揭示动画：宝箱抖动 → 数字滚动 → 最终倍率大字闪光（约 1s）
- 每次落球独立随机，不受难度影响（范围固定 1-100）
- 期望贡献：约 50.5× 均值（用于 RTP 精细校准时以加权分布替代）

---

## 六、历史倍率条

- 位置：Header 正下方，全宽横向滚动
- 展示最近 20 条落槽结果（宝箱显示实际揭示的倍率）
- 颜色编码与倍率槽色阶一致
- 最新结果从右侧插入，向左推移
- 支持手指左右滑动查看更早记录

---

## 七、投注与结算

| 项目 | 说明 |
|-----|------|
| 投注档位 | 1 / 2 / 5 / 10 / 20 / 50 / 100 |
| 结算公式 | 赢得 = 投注额 × 落槽倍率 |
| 宝箱结算 | 赢得 = 投注额 × 随机(1-100) |
| 余额更新 | 实时更新，不足当前投注额时禁止开始 |

---

## 八、AUTO 自动模式

- 点击「自动」按钮：进入自动模式，按钮变金色激活态
- 自动按当前投注额持续投球，无需每球点击开始
- 再次点击自动按钮 / 余额不足 → 退出自动模式

## 十、技术规格

- **渲染：** HTML5 Canvas（钉子板 + 球物理）+ HTML/CSS（UI 层）
- **物理：** 轻量自研（重力 + 钉子碰撞反弹，无需 Matter.js）
- **帧率：** requestAnimationFrame，目标 60fps
- **单球时长：** 约 3-5 秒
- **无后端 MVP：** 前端本地结算，余额本地维护

---

## 十一、MVP 开发范围

**包含：**
- Plinko 物理弹球（12行 × 15槽，Canvas 渲染）
- 穹顶发射入口动画
- 三种难度模式切换（倍率槽动态更新）
- 幸运宝箱随机揭示动画（1-100×）
- 历史倍率条（实时更新，可滑动）
- 投注额选择 + 余额实时计算
- AUTO 自动投注模式
- 开始按钮流程（扣注 → 球落 → 结算 → 更新余额）

**不包含（后续迭代）：**
- 链式分裂球系统
- 累积彩池系统
- 惊喜事件特效
- 音效
- 用户账户 / 真实货币
