# 爆裂足球 7500× · 体验指南

融合 **Plinko 玩法** 与 **裂变倍率机制** 的足球主题 H5 小游戏。  
最高单球倍率 **7500×**，可在PC浏览器或手机端Safari直接体验。

---

## 1️⃣一句话玩法

> 选好行数和难度（一键开启特色功能） → 按 PLAY/自动发球→ 足球碰钉子 → 落入底部倍率槽，**奖金 = 投注 × 倍率**
>
> **PRD文档地址：[https://github.com/brown668/luckydrop/blob/main/PRD.md](https://github.com/brown668/luckydrop/blob/main/PRD.md)**

---

## 2️⃣核心配置

### 支持行数切换（8 – 16 行）

- **行数越多** → 钉子越多、球路越长 → 两端倍率更高

### 三档难度（佛系 / 激进 / 疯狂）


| 难度     | 风险     | 11 行（最大 → 最小） | 16 行（最大 → 最小）    |
| ------ | ------ | ------------- | ---------------- |
| 佛系     | 低      | 9× → 0.9×     | 25× → 0.5×       |
| 激进     | 中高     | 50× → 0.6×    | 250× → 0.3×      |
| **疯狂** | **极高** | 110× → 0.4×   | **2500× → 0.1×** |


> 16 行疯狂 + 高倍足球金球（×3）即可触发游戏标题 **7500×** 单球极值。

## 3️⃣三大特色功能（可单独开启 / 一键全开）

> 顶部「特色功能」按钮（金边胶囊）→ 弹窗内独立开关，亦可点击底部「**爽感拉满**」一键全开。

### ⭐ 裂变足球（Split）  +60% 投注花费

- 球碰到 **金色五角星钉** 时分裂成 **2 颗球**，同步走两条路径，独立结算

### 🔄 免费足球（Respin）  +20%  投注花费

- 球落入 **红色倍率槽** 时，奖励 **1 次免费发球**（不扣余额）
- 免费球同样能触发裂变 / 高倍 / 再次免费
- 强化"撞红球区域 = 滚雪球"的爽感链

### 🔥 高倍足球（hiMult）  +20%  投注花费

- 每次发球随机抽取颜色：
  - **金球** ×3 倍率加成
  - **蓝球** ×2 倍率加成
  - **白球 x1** 普通足球
- 落槽奖金 = 投注 × 槽位倍率 × **球本身倍率**

### 一键全开

- 三个特色全开后总成本 = 投注 × 2.0
- 点击底部"爽感拉满"按钮快速切换

### 自动发球

- 选择自动发球的数量
- 按 PLAY 启动 → 按钮变 STOP，可随时中断
- 进入自动模式后投注/行数/难度/特色锁定，避免误操作

## 4️⃣其他

### 公平性

- 每球结果由 **HMAC-SHA256** 算出（serverSeed + clientSeed + nonce）
- **由于存在分裂球情况，游戏RTP后续需服务端介入进行控制**

### 设置项

- 🎵 BGM 开关（默认开）
- 🔊 音效开关（默认开）
- 📖 游戏规则（4 步教学卡）
- 📊 投注记录
