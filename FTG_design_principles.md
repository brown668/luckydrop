# FTG游戏 — 产品上下文

FTG游戏是一个游戏厂商/品牌。一句话定义：FTG做的是「带彩头的好玩游戏」— 游戏本身好玩是第一位的，Casino结算是附加的彩头。玩得好+运气好，赚钱；技术差或运气差，亏一点。

所有FTG出品的游戏必须同时满足以下6条设计原则：

## 6条核心设计原则

1. 趣味性 — 游戏首先要好玩。通过操作快感、策略深度或叙事体验让玩家获得娱乐。玩家应当觉得自己"在玩游戏"，而非"在赌博"。
2. 能赌钱 — 每款游戏内嵌真金博弈变现模型，具备完整的赔率与概率系统。算法严格控制概率分布、庄家边际与奖金结算，确保公平性可审计。
3. 创新的，两个或多个已有玩法融合 — 拒绝对市面已有品类的简单复制。创新的路径是将已被市场验证的玩法进行杂交组合，产生全新物种。例如：Soulslike + 轮盘赌 = 猎魔轮盘，TD + 赌场 = 塔防赌注。
4. 有意料之外的惊喜 — 游戏过程中必须存在超出玩家预期的时刻，打破常规认知，制造"没想到"的体验。惊喜可以来自隐藏奖励、反转剧情、意外机制触发等。
5. 有持续不断的刺激 — 游戏节奏必须保持紧凑，不允许出现无聊的空窗期。每隔几秒就应有新的决策点、风险变化或反馈，令玩家始终处于肾上腺素分泌的状态。
6. 以PVE为主 — 玩家的对手是系统/庄家，不依赖PVP玩家匹配。这意味着：概率可控、边际可算、无需等待匹配、运营成本低。

## 6条原则的内在逻辑

- 原则3（融合）是实现创新的**方法** — 不从零发明玩法，组合已验证玩法产生新物种
- 原则6（PVE）是实现原则2（赌钱）的**前提** — PVE意味着庄家是系统，概率可控
- 原则1（趣味）是原则2（赌钱）的**包装** — 降低用户心理门槛和监管风险
- 原则4（惊喜）和原则5（刺激）是原则1（趣味）的**两个支柱** — 惊喜制造峰值体验，刺激维持持续参与，二者共同保证"好玩"不是空话

## 设计新游戏时的检查清单

评估任何新游戏方案时，必须逐条验证：
- [ ] 是否好玩？（趣味性）
- [ ] 博弈/变现模型是否清晰？（能赌钱）
- [ ] 是否PVE、不依赖匹配？（PVE为主）
- [ ] 市面上是否已有高度相似的产品？（创新性）
- [ ] 能否明确说出"玩法A + 玩法B"的融合公式？（玩法融合）
- [ ] 是否有让玩家"没想到"的时刻？（惊喜）
- [ ] 游戏节奏是否全程紧凑、无冷场？（刺激）
